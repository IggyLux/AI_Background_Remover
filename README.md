## How to use this tool:

### 1. First time only: Double-click setup.bat and wait for it to finish. (It needs an internet connection to download the AI model and packages).

### 2. Every time after: Double-click start.bat.

### 3. A terminal window will open, and your browser will pop up with the tool.

### 4. Drag an image in, click "Remove Background", and download your PNG.

### 5. Close the browser tab and press CTRL + C in the terminal window when you're done.
